### Exercício Python 053

Crie um programa que leia uma frase qualquer e diga se ela é um **_palíndromo_**, desconsiderando os espaços.

Ex: de frente para tras `==` de tras para frente

- a torre da derrota
- arara
- a sacada da casa

```
Digite uma frase qualquer: a sacada da casa

Esta frase SI es un palíndromo
-------------------------------------------
Digite uma frase qualquer: bola da minha vida

Esta frase NO es un palíndromo
```
