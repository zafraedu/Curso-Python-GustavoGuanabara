Melhore o jogo do desafio 028 onde o computador vai "pensar" em um número entre 0 e 10.
So que agora o jogador vai tentar adivinhar até acertar,
mostrando no final quantos palpites foram necessários para vencer.

```
Digite em um numero de 0 a 10: 7
Tente outro palpite: 3
Tente outro palpite: 4
Tente outro palpite: 1
========================================
Acertou, eu estava pensando no numero 1
Voce levou somente 3 palpites.
```
