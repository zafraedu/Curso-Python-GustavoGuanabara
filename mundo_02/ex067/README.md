### Exercício Python 067

Faça um programa que mostre a tabuada de vários números, um de cada vez, para cada valor digitado pelo usuário.
O programa será interrompido quando o número solicitado for negativo.

```
Input a number: 2
2 X 1  = 2
2 X 2  = 4
2 X 3  = 6
2 X 4  = 8
2 X 5  = 10
2 X 6  = 12
2 X 7  = 14
2 X 8  = 16
2 X 9  = 18
2 X 10 = 20
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-
Input a number: -1
```
