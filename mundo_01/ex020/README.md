### Exercício Python 020

O mesmo professor do desafio anterior quer sortear a ordem de apresentação de trabalhos dos alunos. Faça um programa que
leia o nome dos quatro alunos e mostre a ordem sorteada.

```
Primero aluno: Carlos
Segundo aluno: Maria
Terceiro aluno: Antonio
Quarto aluno: Sara

A ordem de apresentação será
['Carlos', 'Sara', 'Antonio', 'Maria']
```
