### Exercício Python 034

Escreva um programa que pergunte o salário de um funcionário e calcule o valor do seu aumento.
- Para salários superiores a $R\$1250,00$, calcula um aumento de _10%_.
- Para salários inferiores ou iguais, o aumento é de _15%_.

```
Qual é o seu salário? 1000

Teve um aumento de 15%
O valor do seu novo salário  é de R$1150.00
-------------------------------------------
Qual é o seu salário? 1500

Teve um aumento de 10%
O valor do seu novo salário  é de R$1650.00
```
