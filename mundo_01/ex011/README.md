### Exercício Python 011

Faça um programa que leia a largura e a altura de uma parede em metros, calcule a sua área e a quantidade de tinta 
necessária para pintá-la, sabendo que cada **litro** de tinta pinta uma área de **_2m(sqrt)_**

```
Largura: 15
Altura: 2

Será necessario 15.0 litros de tinta para pintar 30.0 metros cuadrados
```
