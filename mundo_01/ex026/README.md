### Exercício Python 026

Faça um programa que leia uma frase pelo teclado e mostre:

- Quantas vezes aparece a letra `"A"`.
- Em que posição ela aparece a primera vez.
- Em que posição ela aparece a última vez.

```
Digite um frase: Uma frase no teclado

A frase tem 3 letras "A".
A primeira letra "A" se encontra na posição 2
A última letra "A" se encontra na posição 17
```
