### Exercício Python 099

Faça um programa que tenha uma função chamada `maior()`, que receba vários parâmetros com valores inteiros.
O seu programa tem que analisar todos os valores e dizer qual deles é o maior.

```
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
Analisando os valores pasados ...
2 9 4 5 7 1 Foram informados 6 valores ao todo.
O maior valor valor informado foi 9.
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
Analisando os valores pasados ...
4 7 0 Foram informados 3 valores ao todo.
O maior valor valor informado foi 7.
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
Analisando os valores pasados ...
1 2 Foram informados 2 valores ao todo.
O maior valor valor informado foi 2.
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
Analisando os valores pasados ...
6 Foram informados 1 valores ao todo.
O maior valor valor informado foi 6.
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
Analisando os valores pasados ...
Foram informados 0 valores ao todo.
O maior valor valor informado foi 0.
```
