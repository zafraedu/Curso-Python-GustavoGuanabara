### Exercício Python 081

 Crie um programa que vai ler vários números e colocar numa lista.
 Depois disso, mostre:

- A) Quantos números foram digitados.
- B) A lista de valores, ordenada de forma decrescente.
- C) Se o valor 5 foi digitado e está ou não na lista.

```
Digite um valor: 5
Quer continual ? [S/N] s

Digite um valor: 4
Quer continual ? [S/N] n

Foram digitados 2 números.
Os valores em ordem decrescente sao [5, 4]
O valor 5 esta na lista
```
