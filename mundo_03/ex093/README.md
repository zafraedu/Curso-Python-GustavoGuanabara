### Exercício Python 093

Crie um programa que gerencie o aproveitamento de um jogador de futebol.

O programa vai ler o nome do jogador e quantas partidas ele jogou.
Depois vai ler a quantidade de gols feitos em cada partida.
No final, tudo isso será guardado num dicionário, incluindo o total de gols feitos durante o campeonato.

```
Nome do Jogador: Zico

Quantas partidas Zico jogou? 4
Quantos gols na partida 0? 3
Quantos gols na partida 1? 1
Quantos gols na partida 2? 2
Quantos gols na partida 3? 0

-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
{'nome': 'Zico', 'gols': [3, 1, 2, 0], 'total': 6}
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
O campo nome tem o valor Zico
O campo gols tem o valor [3, 1, 2, 0]
O campo total tem o valor 6
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
O jogador Zico jogou 4 partidas.
	=> Na partida 0, fez 3 gols.
	=> Na partida 1, fez 1 gols.
	=> Na partida 2, fez 2 gols.
	=> Na partida 3, fez 0 gols.
	
Foi um total de 6 gols.
```
